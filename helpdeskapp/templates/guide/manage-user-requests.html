Managing User Requests

Admin Dashboard Access

As an admin, you can manage all user requests:

  • From Home: Click the "Go to Dashboard" button
  • From Menu: Navigate to "Dashboard" in the main menu
Admin Privilege: You can view and manage all tickets system-wide.
