Delete Tickets

Learn how to safely delete tickets from the system.

Warning: Ticket deletion is permanent and cannot be undone. Use this feature with caution.

When to Delete Tickets

Acceptable & Unacceptable Reasons for Deletion

Delete tickets only in these scenarios:

✓ Valid Reasons:
  • Duplicate tickets
  • Spam tickets
  • Test tickets created during system setup
  • Tickets with personal/sensitive info posted publicly
  • Tickets created by mistake with no content
✗ Invalid Reasons:
  • Ticket is old or resolved
  • User has resolved the issue
  • Ticket is incomplete
  • Ticket is not related to Helpdesk (escalate as needed)
Deletion Process

How to delete a ticket:

  1. Navigate to the ticket - Find the ticket in the dashboard
  2. Click delete button - red "Delete" icon
  3. Confirm deletion - System will ask for confirmation
  4. Verify removal - Check that ticket is no longer in the dashboard
Double-check: Make sure you're deleting the correct ticket - the action cannot be undone.
Documentation Before Deletion

Always document your deletion:

  1. Screenshot the ticket - Capture ticket details before deletion
  2. Record the reason - Why deletion was necessary
  3. Note the date/time - When deletion occurred
  4. Identify the admin - Who performed the deletion
Documentation Template:

Ticket Deletion Log:
• Ticket ID: #1234
• Title: "Spam ticket with spam content"
• Reason: Contained spam links
• Deleted by: Admin Name
• Date: 1 July 2025, 2:30 PM
• Screenshot attached: Yes
